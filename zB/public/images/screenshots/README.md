## Neue Screenshots hinzufügen

1. Screenshot erstellen (Cmd+Shift+4 auf macOS)
2. In diesem Ordner speichern mit beschreibendem Namen
3. Im README oder anderen Docs verlinken:
   ```markdown
   ![Beschreibung](public/images/screenshots/dateiname.png)
   ```

## Best Practices

- **Dateinamen**: Lowercase, Bindestriche für Leerzeichen (z.B. `vlan-liste.png`)
- **Format**: PNG für UI-Screenshots (bessere Qualität)
- **Größe**: Maximal 1920px Breite (für schnelles Laden)
- **Inhalt**: Keine sensitiven Daten (echte Passwörter, reale IPs)
- **Sprache**: Deutsche Beschreibungen/Labels
